# 树莓派调度助手

## 功能描述

nanobot根据配置信息远程连接树莓派，并执行相应的命令，实现智能家居控制、远程运维等功能。

## 核心功能

### 1. 远程连接管理
- **SSH连接**：通过SSH远程连接树莓派
- **多设备支持**：支持配置多个树莓派设备
- **连接信息加密**：敏感信息本地存储

### 2. 命令执行
- **远程命令**：在树莓派上执行Shell命令
- **脚本执行**：运行预定义的自动化脚本
- **结果返回**：获取命令执行结果

### 3. 预设控制命令
- **GPIO控制**：控制树莓派的GPIO引脚
- **摄像头**：拍照或视频监控
- **传感器数据**：读取温湿度等传感器数据
- **服务管理**：重启服务、查看状态等

## 数据存储

数据存储在 `D:\Code\nanobot-xiao\.workspace\data\raspberry\` 目录：
- `devices.json` - 设备配置信息
- `scripts.json` - 预设脚本库
- `logs.json` - 执行日志

## 使用方法

### 添加设备
用户说"添加树莓派"或类似意图时，引导用户提供：
- 设备名称/别名
- IP地址
- SSH端口（默认22）
- 用户名
- 密码或SSH密钥路径

### 执行命令
用户说"在XXX上执行XXX"时，在对应设备上执行命令。

### 预设操作
用户说"开灯"、"拍照"、"查看温度"等时，执行预设命令。

## 命令列表

| 命令 | 说明 |
|-----|------|
| 添加设备 | 添加新的树莓派设备 |
| 列出设备 | 显示所有已配置设备 |
| 删除设备 | 删除设备配置 |
| 执行命令 | 在设备上执行命令 |
| 开灯/关灯 | 控制GPIO灯 |
| 拍照 | 树莓派摄像头拍照 |
| 查看温度 | 读取传感器温度 |

## 技术实现

- 使用 paramiko 库实现SSH连接
- 使用 sshtunnel 处理SSH隧道
- 配置文件存储在本地data目录

## 依赖安装

```bash
pip install paramiko sshtunnel
```
