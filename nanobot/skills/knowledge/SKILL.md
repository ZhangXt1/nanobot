# 个人知识库RAG

## 功能描述

支持向量检索的本地知识库问答，允许用户上传文档并基于文档内容进行智能问答。

## 核心功能

### 1. 文档管理
- **文档导入**：支持PDF、TXT、MD、DOCX等格式
- **文档列表**：查看已导入的文档
- **文档删除**：删除不需要的文档

### 2. 向量存储
- **向量化**：将文档内容转换为向量存储
- **本地存储**：使用chromadb进行本地向量存储
- **增量更新**：支持新增文档而不重建整个知识库

### 3. 智能问答
- **语义搜索**：基于向量相似度检索相关内容
- **问答生成**：结合上下文生成答案
- **引用来源**：标注答案来源的文档

## 数据存储

数据存储在 `D:\Code\nanobot-xiao\.workspace\data\knowledge\` 目录：
- `documents/` - 原始文档存储
- `vectorstore/` - 向量数据库存储
- `metadata.json` - 文档元数据

## 使用方法

### 添加文档
用户说"添加文档"或"导入知识库"时，引导用户提供：
- 文档路径（本地文件）
- 文档名称（可选）

### 问答
用户说"问个问题"或"查询知识库"时，引导用户提供问题。

### 列出文档
用户说"列出知识库文档"时，显示所有已导入的文档。

## 命令列表

| 命令 | 说明 |
|-----|------|
| 添加文档 | 导入新文档到知识库 |
| 列出文档 | 显示知识库中的文档 |
| 删除文档 | 从知识库中删除文档 |
| 搜索 | 搜索知识库相关内容 |
| 问答 | 基于知识库智能问答 |

## 技术实现

- **向量库**：chromadb（支持本地部署）
- **文本分割**：langchain.text_splitter
- **嵌入模型**：sentence-transformers
- **LLM调用**：集成现有LLM接口

## 依赖安装

```bash
pip install chromadb langchain langchain-community sentence-transformers pypdf python-docx
```

## 配置

可在 `config.json` 中配置：
- 嵌入模型选择
- 文本分块大小
- 检索结果数量
